# GitHubReleaseDownloads
GitHubのReleasesにあるSource codeを除くAssetのDL数を表示します

## Installation
1. Greesemonkeyまたはその派生(推奨はViolentmonkey)をインストールする
2. ReleaseにあるGitHubReleaseDownloads.user.jsをクリックする
